Notes: all three columns render the same labels at the same sizes with the same plate styling. AvQest is loaded locally via @font-face from samples/AvQest.ttf; Cinzel and Cinzel Decorative come from Google Fonts. The bundled 1001fonts-avqest-eula.txt (FFC license) explicitly permits commercial web embedding (§2, §6) and woff2 conversion (§4) — so AvQest is cleared for the deployed editor. If AvQest looks right, that's our font. Cinzel stays as a fallback for browsers that fail to load the woff2.
