Indicator lane sits right of the preview — never overlaps. Always 24×44 with two fixed 18×18 slots: top for sound, bottom for minimap. Sound is binary (speaker icon present = rule plays any alert sound; absent = none — the specific sound is set in the rule detail, not surfaced here). Minimap slot shows only a colored dot whose color matches the rule's minimap icon color. Rows with neither still render the empty lane so the column stays aligned.

A future refinement: hovering the speaker could play the sound and the minimap dot could show the icon's actual in-game shape once the engine-side dataset is available.
